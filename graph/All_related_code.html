- RIDGE LINE -

Ridgeline

Allows to compare the distribution of a few variables. Usefull if there is a pattern to observe
- BOXPLOT -

Boxplot

Summarize the distribution of several numeric variables using boxes. Use it if you have a limited dataset size.
- VIOLIN -

Violin

Show the density of several numeric variables. Use it with a large amount of data
- SCATTER PLOT -

Scatter plot

Display the relationship between 2 numeric variables
- BUBBLE PLOT -

Bubble plot

Add a third dimension to your scatter: the circle size
- CORRELOGRAM -

Correlogram

Shows the relationship between each pair of numeric variables
- CONNECTED SCATTER-

Connected Scatterplot

Very close from a scatterplot, but link data points with segments
- DENSITY 2D -

Density 2d

One of the best way to avoid overplotting for big sample size
- BARPLOT -

Barplot

Represents the value of entities using bar of various length.
- SPIDER -

Spider / Radar

One or more series of values over multiple common quantitative variables. Axis are joint on the middle of the figure.
- WORDCLOUD -

Wordcloud

A visual representation of text data, where word size is relative to their frequency.
- PARALLEL -

Parallel coord.

Compare samples for several numeric values.
- LOLLIPOP-

Lollipop plot

A fancy barplot where bars are replaced with a dot and a segment. Try it and adopt it.
- CIRCULAR BAR -

Circular Barplot

A barplot with a circularized X axis. Useful if you have many groups to display with obvious patterns.
- TREEMAP -

Treemap

Displays hierarchical data as a set of nested rectangles. Perfect to show how the whole is divided.
- VENN DIAGRAM-

Venn diagram

Shows all possible logical relationships between a finite collection of different sets
